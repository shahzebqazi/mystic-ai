# Desktop UX mockups (showcase)

Interactive **desktop-shell** prototypes: widget layout, brand tokens, static marketing pages. Part of the [mystic-ai](https://github.com/shahzebqazi/mystic-ai) design showcase; wireframe notes in `src/imports/pasted_text/dotai-wireframes.md`.

## Local development

```bash
npm install
npx vite
```

## GitHub Pages

The app is deployed via the **Deploy Mockups to GitHub Pages** workflow.

- **Settings > Pages:** Set **Source** to **GitHub Actions** (not "Deploy from a branch"). The workflow builds this app and deploys the artifact.
- Site URL: https://shahzebqazi.github.io/mystic-ai/
